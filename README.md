# TimeCalculation

Given a parameter n, and a list l, find all the combinations that are closest to n that don't go over.

Example:

n= 50
l= [25, 10, 30]

Returns:
(10, 30) -> 40
(10, 10, 30) -> 50
(10, 10, 10, 10) -> 40
(25, 25) -> 50
(10, 25) -> 35
(30,) -> 30
(10, 10, 25) -> 45
(10, 10, 10) -> 30
(10, 10, 10, 10, 10) -> 50
